# ranobehub
**ranobehub** – это модуль системы управления парсерами [Melon](https://github.com/otaku-melons/Melon), включающий поддержку источника: [RanobeHub](https://ranobehub.org).

## Коллекция
Таблица поддерживаемых ключей для `melon collect`.
| Ключ | Поддержка | Описание |
|---|---|---|
| **&#x2011;&#x2011;period** | ✅ | Период, за который нужно получить обновления (в часах). |
| **&#x2011;&#x2011;filters** | ✅ | Параметры запроса из URI страницы каталога. |
| **&#x2011;&#x2011;pages** | ✅ | Количество страниц каталога, с которых нужно получить данные. |
